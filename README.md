# Laravel Blade Plugin for aCode

Advanced Laravel Blade support for **aCode Editor (Android)**.

## Features
- Blade syntax highlight
- Autocomplete directives
- Snippets (@if, @foreach, @section, etc)
- Supports `.blade.php`
